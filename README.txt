
블록체인 부동산 디지털화 플랫폼 예시 (V1 설계 + 프로토타입)

이 폴더에는 다음이 포함되어 있습니다.
- index.html : 전체 화면이 들어 있는 단일 페이지 애플리케이션(SPA) 예시
- styles.css : 기본 스타일
- app.js : 예시 데이터, 네비게이션, 지갑 연결(메타마스크) 기본 코드
- /contracts : Solidity 스마트 계약 예시 코드

이 예시는 '사업용 기획 및 기술 구조'를 한 번에 볼 수 있도록 하는 목적으로 작성되었습니다.
실제 서비스로 사용하려면 다음이 추가되어야 합니다.
- 백엔드 서버 및 데이터베이스
- 스마트 계약 실제 배포 및 검증
- 한국/해외 부동산 및 금융 관련 법규 검토
- KYC/AML, 전자금융거래법, 자본시장법 등 규제 검토

GitHub Pages에 올리는 간단한 방법 (정식 배포 아님):
1. GitHub에 새 저장소(repository)를 만듭니다.
2. 이 폴더의 파일들을 그대로 업로드합니다.
3. 저장소 설정(Settings)에서 "Pages" 항목을 찾아 branch를 main, / (root)로 선택합니다.
4. 배포 주소(URL)가 생성되면 브라우저에서 접속할 수 있습니다.

※ Solidity 컨트랙트는 /contracts 폴더의 파일을 Remix, Hardhat, Foundry 등 툴에서 열어 테스트넷에 배포할 수 있습니다.
